기능 구현 목록
============
> 자동차 이름 입력 기능
> + 쉼표 기준으로 자동차를 나눔
> + [예외처리] 알파벳이 아닌 자동차 이름
> + [예외처리] 같은 자동차 이름을 입력
> + [예외처리] 이름 길이가 5를 초과
> + [예외처리] 구분자가 자동차 이름 사이의 ","가 아님

> 시도할 회수 입력 기능
> + 자연수가 아닌 경우

> 자동차 전진 기능
> + 각 자동차에 대한 난수 발생
> + 4 이상일 때만 전진
> + 자동차가 전진하면 랭킹 최신화

> 우승자 선별 기능
> + 인덱스 순회하며 랭킹의 최대값과 맞는 랭킹 원소를 확인
> + 우승자 리스트에 최대값의 랭킹을 가지고 있는 자동차 이름 삽입
